# 自检循环规范

供 SKILL.md 第 9 步调用。定义 Prompt 生成后的自动自检与回修逻辑。

---

## 触发时机

Prompt 生成完毕后**立即**执行自检，不等待用户确认。

## 循环逻辑

```
生成 Prompt
    ↓
执行自检（对照 wild-art-spec.md，同目录）
    ↓
分类结果：
  ❌ 不合规项 → 自动修正 → 重新生成 → 再次自检
  ⚠️ 关注项   → 记录到报告（不触发循环，生图后验证）
  ✅ 通过项   → 记录到报告
    ↓
循环继续直到：仅剩 ✅ 和 ⚠️ 项
    ↓
进入自动存档
```

## 循环上限

- **最大循环次数：3 轮**
- 超过 3 轮仍有 ❌ 项：输出警告，标记未解决项，进入存档（需人工介入）

---

## ❌ 强制纠正规则

以下项检测到不合规时，**自动修正，不询问用户**：

| 检查项 | 不合规判定 | 自动纠正措施 |
|-------|-----------|-------------|
| 主体视角 | 非正面视角 | 添加 `front-facing view / 正面视角` |
| 边框类型 | 非单线 / 出现双线描述 | 修正为 `thin single-line / 纤细单线`，移除双线描述 |
| 边框宽度 | 未提及或 >4% | 添加 `2-4% of canvas width / 画布宽度2-4%` |
| 边框形状 | 未明确写入 | 根据用户 Q26 答案添加形状描述 |
| 核心色数量 | >3 种 | 添加 `max 3 core colors / 最多3种核心色` |
| 主色饱和度 | 未提及 | 添加 `saturation ≥65% / 饱和度≥65%` |
| WILD 文字位置 | 非底部位置 | 添加 `at bottom quarter / 位于底部四分之一` |
| WILD 文字立体感 | 无 3D/浮雕描述 | 添加 `3D embossed relief / 3D浮雕立体感` |
| 外发光 | 无外发光描述 | 添加 `outer glow / 外发光` |
| WILD 文字超框 | 未描述超框 | 添加 `overflows frame 15-25% / 超框15-25%` |
| 主体占比 | 未提及或不在 75-95% | 添加 `subject 75-95% / 主体占比75-95%` |

---

## ⚠️ 关注项（记录但不循环）

以下项仅靠 Prompt 文本无法完全验证，记录到报告供生图后人工验证：

- 主体与边框超越比例是否视觉协调
- 色彩搭配实际效果
- WILD 文字可读性（64px 最小尺寸）
- 整体视觉层级是否突出
- 画布背景色渲染效果

---

## 自检报告格式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
美术规范自检报告
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第 N 轮自检】（N=1,2,3...）

✅ 通过项：
  - [项目名]：[检查结果]
  ...

⚠️ 关注项（生图后验证）：
  - [项目名]：[原因]
  ...

❌ 不合规项（已自动修正）：
  - [项目名]：[原值] → [修正值]
  ...

【最终状态】全部 ✅/⚠️，进入存档
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
