# 质量控制规范
# 供 SKILL.md 第 7 步（自检循环）调用，也供人工评审使用。
# 自检逻辑以 `wild-art-core.md`（核心美术约束12条）为判定依据。

---

## 自检循环规则

- **触发时机**：Prompt 生成完毕后立即执行，不等待用户确认
- **最大轮次**：3 轮；超过仍有 ❌ 则输出警告并标记未解决项
- **循环逻辑**：检查 → 分类 → 自动修正 ❌ → 重新生成 → 再次检查，直到仅剩 ✅ 和 ⚠️

---

## ❌ 强制纠正规则（文本可验证，自动修正）

| 检查项 | 不合规判定 | 自动纠正措施 |
|-------|-----------|-------------|
| 主体视角 | 无正面视角描述 | 添加 `front-facing view / 正面视角，直视观众` |
| 边框类型 | 非单线 / 出现双线描述 | 修正为 `thin single-line / 纤细单线`，移除双线描述 |
| 边框宽度 | 未提及或 >4% | 添加 `2-4% of canvas width / 画布宽度2-4%` |
| 边框形状 | 未明确写入 | 根据 Q26 答案添加形状描述 |
| 核心色数量 | >3 种描述 | 添加 `max 3 core colors / 最多3种核心色` |
| 主色饱和度 | 未提及 | 添加 `saturation ≥65% / 饱和度≥65%` |
| WILD 文字位置 | 非底部位置描述 | 添加 `at bottom quarter, 70-85% from top / 位于底部四分之一` |
| WILD 文字立体感 | 无 3D/浮雕描述 | 添加 `3D embossed gold relief / 3D浮雕立体金色` |
| 外发光 | 无外发光描述 | 添加 `outer glow, non-white / 外发光，非纯白` |
| WILD 文字超框 | 未描述超框 | 添加 `overflows frame 15-25% downward, 2-5% wider / 向下超框15-25%，横向超2-5%` |
| 主体占比 | 未提及或不在 75-95% | 添加 `subject 75-95% of symbol height / 主体占符号格75-95%` |

## ⚠️ 关注项（仅生图后可验证，记录到报告但不触发循环）

- 主体与边框超越比例是否视觉协调
- 色彩搭配实际效果（饱和度、对比度）
- WILD 文字 64px 最小尺寸可读性
- 整体视觉层级是否突出
- 皮肤/布料哑光质感（无油光、无塑料感）
- 画布背景色渲染效果

## 人工评审清单（生图后使用）

> 拿到生图结果后对照以下各项逐一核查。

### 视觉层级
- [ ] WILD 是卷轴中视觉重量最高的图标（或符合其定位层级）
- [ ] 80px 缩略图下仍能一眼认出是 WILD
- [ ] 主色调与 SCATTER/BONUS 有明显区分（色相差 ≥ 30°）

### 构图
- [ ] 主体占符号格高度 75-95%
- [ ] 四边安全边距 ≥ 画布尺寸的 5%
- [ ] 主体视觉重心略高于中心（Y 轴约 45-50%）
- [ ] 主体完整，无裁切

### 色彩
- [ ] 核心识别色不超过 3 种
- [ ] 主色饱和度（HSB S 值）≥ 65%
- [ ] 主体最亮区域与背景对比度 ≥ 4:1（如背景已知）

### 皮肤/材质
- [ ] 皮肤/布料哑光低光泽（无油光、无塑料高光）
- [ ] 金属/宝石等例外材质高光自然

### 发光
- [ ] 存在外发光（Outer Glow）
- [ ] 外发光颜色与主色有色相关联（非纯白）

### 边框
- [ ] 边框为单线，宽度 2-4%（如有边框）
- [ ] 边框形状与 Q26 选择一致
- [ ] 无边框时：轮廓有描边或边缘光与背景分离

### WILD 文字
- [ ] 文字位于底部四分之一区域
- [ ] 向下超出边框下缘 15-25%
- [ ] 两端横向各超出边框约 2-5%
- [ ] 3D 浮雕立体感（有厚度/高光/阴影）
- [ ] 64px 下至少可辨认 3 个字母

### 技术规格
- [ ] 格式为 PNG-32（带 Alpha 通道）
- [ ] 色彩空间为 sRGB
- [ ] 文件大小合理（512×512 建议 < 800KB）
