# 核心美术约束（12条）

自检时逐条验证。文本可验证的 ❌ 项自动修正，仅生图后可验证的 ⚠️ 项记录到报告。详细自检表格见 `quality-control.md`（按需加载）。

1. **主体视角**：正面视角（front-facing view / 直视镜头），禁止四分之三或侧面视角
2. **主体占比**：占符号格高度 75-95%；视觉重心略高于画布中心（Y 轴 42-50%）
3. **主体超框**：头部/焦点元素微超边框上缘 3-8%；主体图层置于边框图层之上（Z 轴顺序）
4. **安全边距**：四边 ≥ 5% 画布尺寸（外发光可超出，主体轮廓不可超出）
5. **视觉层级**：WILD 视觉重量最高（或符合其层级定位）；80px 缩略图下首先被注意到
6. **色彩**：核心色 ≤ 3 种，主色饱和度 ≥ 65%（HSB S 值）；WILD 与 SCATTER 色相差 ≥ 30°；禁纯灰主体
7. **皮肤/布料**：哑光低光泽（specularity < 20%，近零镜面反射），皮肤次表面散射；禁油光/强反光（金属/宝石/配饰例外）
8. **外发光**：必须存在；禁纯白色；颜色与主色有色相关联
9. **边框**：单线（Single Line），宽度 2-4% 画布；禁双线/厚框/多线并排；形状按 Q26 答案明确写入
10. **框内背景**：主题暗色氛围，强明暗对比；禁花哨纹样、禁白色/亮色/高饱和背景
11. **WILD 文字位置**：位于底部 1/4 区域（距顶 70-85%），向下超框 15-25%，横向两端各超框 2-5%
12. **WILD 文字外观**：3D 浮雕立体金色，有厚度侧面+高光面+阴影面至少 2 项；禁平面 2D；字体融入主题元素（如北欧符文刻痕/龙鳞纹理/熔岩裂纹）；字距非负值；64px 下可辨 3 个字母
