# 联网搜索与智能分析策略

供 SKILL.md 第 3-5 步调用。定义搜索维度、分析方法与智能选项生成逻辑。

---

## 触发时机

第一阶段 Batch 1.4（Q13 平台选择）回答完毕后，**立即**并行执行。

## Step 1：并行搜索

### 竞品 WILD 图标分析（3 条搜索）

```
WebSearch: "[游戏主题] slot machine WILD icon [工作室/风格参考] 2026"
WebSearch: "老虎机 WILD 图标 [竞品名称] 设计风格分析"
WebSearch: "casino game WILD symbol design [目标市场] market"
```

**必须获取的维度：**

| 维度 | 要求 |
|------|------|
| 竞品数量 | ≥ 3 个竞品的 WILD 图标特点 |
| 构图趋势 | 对称/垂直/放射等 |
| 色彩偏好 | 主色/辅助色 |
| 边框样式 | 细线/粗框/无边框 |
| 文字处理 | 位置/字体风格 |

### AI 模型能力与 Prompt 规范（每平台 3 条搜索）

```
WebSearch: "[平台名] image generation latest models [当前年份] features"
WebSearch: "[平台名] 最新图像生成模型 Prompt技巧 [当前年份]"
WebSearch: "[平台名] AI生图 模型能力 美术风格 参数 [当前年份]"
```

**每平台必须获取：**

| 维度 | 要求 |
|------|------|
| 模型版本 | ≥ 3 个可用模型版本号与发布日期 |
| 语言偏好 | 中文/英文/双语 |
| 结构方式 | 关键词/自然语言/结构化段落 |
| 关键参数 | 专属参数与语法 |
| 风格擅长 | 美术风格擅长领域 |
| 局限性 | 已知弱点与规避方法 |

## Step 3：智能分析与选项生成

基于竞品分析 + AI 模型能力，为第三阶段问卷每题生成 **智能推荐选项**（非固定模板）。

### 分析维度 → 选项生成映射

| 分析维度 | 输入数据 | 生成目标 |
|---------|---------|---------|
| 世界观氛围 | Q1(主题) + Q3(市场) + 竞品分析 | 推荐氛围关键词 / 避免氛围 / 色调倾向 |
| 美术风格 | Q2(玩家) + Q12(复杂度) + 竞品 + AI模型擅长风格 | Q15 智能选项 |
| 材质与光照 | Q1(主题) + 竞品分析 | Q16、Q17 智能选项 |
| 色彩 | Q1(主题) + 竞品主色 | Q19 智能选项 |
| WILD 文字 | Q1(主题) + 竞品文字处理方式 | Q21、Q22 智能选项 |
| 边框与背景 | 竞品分析 + `wild-art-spec.md` 规范 | Q25、Q26、Q27 智能选项 |

### 选项格式要求

- 每题 3-4 个选项
- 推荐项标注理由（如："推荐 — 竞品趋势+AI模型擅长"）
- 其他选项标注适用场景
