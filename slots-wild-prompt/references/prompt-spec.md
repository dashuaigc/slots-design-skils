# Prompt 生成规范

供 SKILL.md 第 8 步调用。定义多平台差异化 Prompt 的生成规则。

---

## 核心原则

- 对每个平台/模型**分别生成独立 Prompt**，禁止简单互译或复制外壳。
- **每个平台只输出该平台所需的语言版本**（不再双语对照）。

---

## 平台专项约束与输出语言

| 平台 | 输出语言 | 文件格式 | 约束 |
|------|---------|---------|------|
| **Midjourney** | 仅英文 | `.md` | 建议精炼在 60-80 词 |
| **Nano Banana** | 中英文双版 | `.md` | 结构化JSON片段，无硬字数限制 |
| **即梦** | 仅中文 | `.md` | 总字数**不超过 800 字**；超出时按优先级删减 |
| **其他平台** | 按平台偏好 | `.md` | 参考 `ai-models/` 对应模型规范文件 |

---

## 内容覆盖清单

每条 Prompt **必须**覆盖以下全部维度：

| 维度 | 说明 |
|------|------|
| 主体 | 主体物描述、形态、占比 |
| 风格 | 绘画风格、写实等级 |
| 材质 | 材质类型、粗糙度/金属度倾向 |
| 光照 | 光照风格、方向、强度 |
| 配色 | 主色调、辅助色、饱和度区间 |
| 构图 | 视角、主体位置、层次关系 |
| WILD 文字 | 字体风格、位置、立体感、装饰、超框描述 |
| 边框 | 样式、形状、宽度 |
| 背景 | 框内背景处理、画布背景色 |
| 负向限制 | 明确排除的元素 |
| 技术约束 | 分辨率、比例、参数 |

---

## 质量匹配维度

每个 Prompt 必须满足：

| 匹配维度 | 要求 |
|---------|------|
| 语言匹配 | 使用该模型最优输入语言 |
| 结构匹配 | 适配该模型偏好的 Prompt 结构方式 |
| 参数匹配 | 正确使用该模型专属参数与语法 |
| 特性利用 | 针对该模型优势重点刻画 |
| 弱点补偿 | 针对已知局限做补偿性描述 |
| 竞品融合 | 融入联网搜索的竞品趋势分析 |

---

## 风格量化表达要求

风格描写禁止笼统，必须量化：

- 写实等级（如：semi-realistic / hyper-realistic）
- 细节可读度（如：readable at 64px / rich detail at 512px）
- 材质粗糙度/金属度倾向
- 边缘锐度（如：sharp edges / soft diffused edges）
- 局部对比
- 饱和度区间

---

## 美术规范强制写入项

以下内容**必须**写入每条 Prompt（规则来源：SKILL.md 核心美术约束，无需重复加载 wild-art-spec.md）：

| 规范项 | Prompt 写入要求（EN / CN） |
|-------|--------------------------|
| 主体正面视角 | front-facing view / 正面视角 |
| 主体占比 75-95% | subject occupies 75-95% / 主体占比75-95% |
| 主体微超边框 3-8% | subject protrudes 3-8% above frame / 主体微超边框上缘3-8% |
| 边框单线细窄 2-4% | thin single-line frame, 2-4% canvas width / 纤细单线边框，画布宽度2-4% |
| 边框形状 | 按用户 Q26 答案写入形状 |
| 框内背景 | dark [theme] atmospheric background, strong contrast / 主题暗色氛围背景，强明暗对比 |
| WILD 文字底部位置 | "WILD" text at bottom quarter / WILD文字位于底部四分之一 |
| WILD 文字超边框 | text overflows frame 15-25% downward, 2-5% wider / 文字向下超框15-25%，宽度略超2-5% |
| WILD 文字立体感 | 3D embossed gold / 3D浮雕立体金色 |
| WILD 字体组合 | 按用户 Q21+Q23 组合描述 |
| 文字关联装饰 | 按用户 Q24 选择写入 |
| 画布背景色 | 按用户 Q7 答案写入 |
| 核心色≤3种 | max 3 core colors / 最多3种核心色 |
| 主色饱和度≥65% | saturation ≥65% / 饱和度≥65% |
| 外发光 | outer glow (non-pure-white) / 外发光（非纯白） |

---

## WILD 字体融合规则

WILD 字体**必须**融入游戏主题元素，禁止泛用空泛字体描述：

- 融入主题纹样（如：古埃及象形纹、北欧符文刻痕）
- 融入铭刻符号（如：龙鳞纹理、宝石镶嵌）
- 融入材质语言（如：熔岩裂纹、冰晶折射）

## 竞品信息融合

若用户提供竞品信息或联网搜索获得竞品分析：
- 拆分"可借鉴点"和"避免点"
- 分别写入 Prompt 的正向描述和负向限制
