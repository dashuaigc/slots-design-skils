# SCATTER 图标美术规范
## Slots Game SCATTER Symbol Art Specification

版本：1.0 | 适用范围：所有平台老虎机游戏 SCATTER 图标静态美术资产

---

## 规范使用说明

本文档定义了 SCATTER 图标美术的行业通用标准，在以下阶段均应对照本规范：
1. 生成 AI Prompt 前（设计规划阶段）
2. 生成 AI 图像后（人工评审阶段）
3. 精修完成后（交付前终检阶段）

---

## 第零章：预览画布规范

评审用画布背景色与 WILD 规范一致：
- 标准中灰：`#666666`（评审图标轮廓与外发光清晰度）
- 浅中灰：`#999999`（评审图标暗部细节可见性）

> 最终交付仍为透明背景 PNG-32，此规范仅用于设计过程评审。

---

## 第一章：视觉层级规范

### 1.1 SCATTER 的符号层级定义

**核心原则：SCATTER 是老虎机中触发最高奖励（免费旋转/奖励轮）的符号，其视觉重量必须不低于 WILD，且通常应更抢眼。**

| 符号类型 | 视觉层级 | 视觉重量基准 |
|---------|---------|------------|
| SCATTER 图标 | **最高** | **100%** |
| WILD 图标 | 最高 | 95-100%（与SCATTER并列或略低） |
| 高价值主题图标（High Symbols） | 中高 | 55-70% |
| 低价值图标（A/K/Q/J/10） | 最低 | 30-45% |

**规范要求：**
- SCATTER 的综合视觉重量（发光强度 + 色彩饱和度 + 独特色相）必须不低于 WILD
- 在卷轴缩略视图（约 80×80px）中，SCATTER 应是第一眼被注意到的图标
- **实现方式：通过色相对比、更大外发光、更高饱和度达成视觉超越，而非依靠主体尺寸**

### 1.2 与 WILD 的视觉区分策略

SCATTER 图标必须与 WILD 保持清晰的视觉区分，策略优先级：

| 区分维度 | 要求 | 实现方法 |
|---------|------|---------|
| **色相差异** | 与 WILD 主色色相差 ≥ 30°（强制） | 选择与 WILD 完全不同的主色 |
| **外发光强度** | SCATTER 外发光强度 ≥ WILD | 增大发光半径或不透明度 |
| **形态差异** | 形状语言明显不同 | WILD 是人物 → SCATTER 选宝石/宝物 |
| **特效差异** | SCATTER 有内发光/折射，WILD 以外发光为主 | 宝石类内外并发 |

## 第二章：构图规范

### 2.1 主体占比

```
主体元素（含边框）安全占比：65% – 85% 的画布面积
视觉重心位置：略高于画布中心（Y 轴 42-50% 位置）
注：比 WILD 的 75-95% 放宽下限，SCATTER 主体（如宝石/宝箱）允许稍小
```

### 2.2 主体超出边框规范

```
超出量：0-8%（可选，比 WILD 更保守）
宝石/宝珠类：超出部位为顶点或棱角（非头部）
人物/神明类：参考 WILD 规范（头部超框 3-8%）
超出部分：仍在画布安全边距内（不超出画布边缘）
图层逻辑：主体图层置于边框图层之上
```

### 2.3 安全边距

```
最小边距（四边）：画布尺寸的 5%
推荐边距（四边）：画布尺寸的 8-10%
```

发光效果可超出安全边距，主体轮廓线不能超出。

### 2.4 构图视角

| 主体类型 | 推荐视角 | 说明 |
|---------|---------|------|
| 宝石/水晶/多面体 | 3/4 视角或微俯视 ⭐ | 展示切割面与折射效果，识别度最高 |
| 宝球/水晶球 | 正面视角 | 圆形主体，正面即最佳 |
| 人物/神明/神兽 | 正面视角（front-facing） | 同 WILD 规范 |
| 宝箱/符文石 | 3/4 视角或正面 | 展示立体感与纹理 |

## 第三章：色彩规范

### 3.1 核心识别色数量

```
主色调：1 种（占主体面积 50% 以上）
辅助色：最多 2 种
高光/发光色：1 种
总计：核心色不超过 3 种
```

### 3.2 与 WILD 的色相差规范（核心规则）

```
强制规则：SCATTER 主色与同套 WILD 图标主色的色相差 ≥ 30°

常见组合参考：
  WILD 金色（40-60°）→ SCATTER 蓝紫（220-280°）    差约 170°  ✅
  WILD 红色（0-20°）  → SCATTER 蓝绿（160-200°）    差约 170°  ✅
  WILD 绿色（120°）   → SCATTER 紫色（280-300°）    差约 160°  ✅
  WILD 金色（50°）    → SCATTER 橙红（10-20°）      差约 30°   ⚠️ 最小合规，不推荐
```

### 3.3 色彩饱和度要求

```
SCATTER 主色饱和度（HSB S 值）应不低于 65%
目标：在 #666666 背景上，SCATTER 的颜色应比 WILD 更鲜明夺目
禁止：纯灰色主体（无色彩感）
```

### 3.4 宝石/水晶类材质的色彩特殊规则

宝石类主体允许以下 WILD 规范中禁止的处理方式：
- ✅ 高光反射点（模拟切割面受光）
- ✅ 多点光源（多面体的各切割面各有高光）
- ✅ 内发光（模拟宝石内部晶体发光）
- ✅ 彩虹折射微渐变（如钻石光谱）

## 第四章：发光规范

### 4.1 外发光（与 WILD 同级或更强）

```
外发光强度规范：
  SCATTER 外发光综合效果（面积×强度）不低于同套 WILD 图标

参数参考（@512px 画布）：
  标准外发光（窄）：
    · 模糊半径：12-20px
    · 颜色：主色亮化版（HSB 亮度 +30-50%）
    · 不透明度：70-90%
    · 混合模式：Linear Dodge (Add)

  大范围光晕（宽）：
    · 模糊半径：40-60px
    · 颜色：主色调
    · 不透明度：30-50%
    · 混合模式：Linear Dodge (Add)
```

**外发光颜色规范：**
- 禁止纯白色外发光
- 外发光颜色与主色有色相关联
- 宝石类：外发光颜色应与宝石主色匹配

### 4.2 内发光（宝石/水晶类特有）

```
适用场景：宝石/水晶/宝珠/魔法球等透光材质

内部光源核心：
  · 模糊半径：5-10px
  · 不透明度：80-100%

内部光源漫射：
  · 模糊半径：15-25px
  · 不透明度：40-60%
```

### 4.3 多点光源（多面切割宝石专用）

```
多点高亮：模拟宝石各切割面受光
  · 每个高光点：4-8px 模糊半径
  · 颜色：比主色亮 50-70%（接近白色的主色亮化版）
  · 不透明度：60-80%
  · 分布：沿切割面棱线或顶角

注意：多点高亮不替代外发光，两者并存
```

## 第五章：边框规范

与 WILD 规范完全一致：

```
边框宽度：画布尺寸的 2-4%（严格限制）
类型：单线（Single Line）— 禁止双线/多线并排/厚框
```

### 5.1 边框形状偏向

SCATTER 图标边框形状选择上，偏向圆形/椭圆/圆角矩形（与宝石/宝珠主体形态呼应）：

| 形状 | 适用主体 | 推荐优先级 |
|------|---------|---------|
| **圆形** | 宝球/水晶球/星形 | ⭐⭐⭐ 优先推荐 |
| **椭圆形** | 蛋形宝石/垂直构图宝珠 | ⭐⭐⭐ |
| **圆角矩形** | 宝箱/方形宝石/多数主体 | ⭐⭐ 通用选择 |
| **矩形** | 人物/神明/多数主体 | ⭐ 最通用，但不如圆形匹配SCATTER |
| **其他多边形** | 特殊主题（菱形宝石/盾形） | 按主题决定 |

### 5.2 框内背景

```
核心要求：主体与背景形成清晰明暗对比
宝石/水晶类特殊允许：极简深色单色底（宝石本身足够复杂，背景退让）
禁止：花哨纹样、白色/亮色/高饱和背景
```

## 第六章：SCATTER 标签文字规范

### 6.1 文字内容选择

| 内容类型 | 适用场景 | 注意事项 |
|---------|---------|---------|
| "SCATTER" | 通用，功能清晰 | 7个字母，字距需仔细控制 |
| "FREE SPINS" | SCATTER专触发免费旋转时 | 较长，字号需缩小 |
| 主题词 | 强主题沉浸感 | 需确保功能可识别 |
| 无文字 | 极简/像素风/高识别主体 | 主体形态必须足够独特 |

### 6.2 文字位置规范（有文字时）

```
文字垂直位置：底部 1/4 区域（距顶 70-85%）
向下超框量：文字高度的 10-25%（向下超出边框下缘）
文字水平宽度：略超出边框内宽，两端各超出约 2-5%
文字水平位置：水平居中

注："SCATTER" 比 "WILD" 更宽，字号相同时横向占比更大，
   建议字距保持标准或略收窄（但不为负值）
```

### 6.3 文字颜色规范（核心差异于 WILD）

**SCATTER 标签颜色不默认金色，由问卷 Q21_S 决定：**

| 颜色选项 | HSB 参考值 | 适用主题 |
|---------|---------|---------|
| 暖金色 | H:40-50°, S:80-95%, B:80-100% | 财富/喜庆/古典 |
| 冷银色 | H:200-220°, S:5-20%, B:85-100% | 科技/现代/月亮 |
| 主题紫蓝能量色 | H:220-300°, S:60-90%, B:80-100% | 魔法/神秘/宇宙 |

**所有颜色均须满足：**
- 3D 浮雕立体感（有厚度侧面+高光面+阴影面至少 2 项）
- 禁止平面 2D 文字
- 字距非负值

### 6.4 文字立体感要求

与 WILD 规范一致：
- 必须包含：厚度面（Side faces）+ 高光面（Highlight）+ 阴影面（Shadow）至少 2 项
- 浮雕感（Bas-relief）：文字从背景表面凸起
- 禁止：仅有投影（Drop shadow）但无立体厚度的 2D 文字

---

## 第七章：多尺寸适配规范

与 WILD 规范一致：

| 显示尺寸 | 可读性要求 |
|---------|---------|
| ≥ 128×128px | 所有细节清晰，文字（若有）完全可读 |
| 80-128px | 主体轮廓清晰，文字可辨认 |
| 64×64px | 主体形态可识别，能判断是 SCATTER |
| < 64px | 外发光 + 整体形态剪影可辨 |

---

## 第八章：技术交付规范

与 WILD 规范一致：
- 格式：PNG-32（带 Alpha 通道）
- 色彩空间：sRGB
- 标准尺寸：256/512/1024px（推荐 512px）

**文件命名：**
```
scatter_[主题标识]_[尺寸]_[状态].png

示例：
  scatter_caishen_512_normal.png
  scatter_caishen_512_highlight.png
```
