## SCATTER 专项人工检查清单

生图后必须核查以下项（无法在 Prompt 中预防）：

**视觉层级检查：**
- [ ] 将 SCATTER 和 WILD 缩略图并排放置（80×80px），SCATTER 是否更抢眼？
- [ ] SCATTER 外发光面积/强度是否 ≥ WILD？
- [ ] SCATTER 主色是否比 WILD 更饱和或对比更强？

**色相差检查：**
- [ ] 在取色工具中分别取 SCATTER 和 WILD 主色的 Hue 值，差值是否 ≥ 30°？

**宝石折射检查（器物类）：**
- [ ] 宝石各切割面是否有独立高光点（而非单一高光）？
- [ ] 内发光是否存在，且颜色与外发光协调？

**文字检查（有文字时）：**
- [ ] 文字是否在底部1/4区域，并向下超出边框？
- [ ] 文字颜色是否与 Q21_S 选择一致（不一定是金色）？
- [ ] 文字是否有 3D 立体浮雕感（有侧面厚度）？
- [ ] 64px 下文字是否可辨认？

**整体效果：**
- [ ] 与同套 WILD 图标并排，视觉风格是否统一（同系列感）？
- [ ] 在 #666666 和 #999999 背景下分别检查，两种背景下均清晰可见？
