## Step 1：并行搜索

### 1.1 竞品 SCATTER 图标分析（3 条搜索）

```
WebSearch: "[游戏主题] slot machine SCATTER symbol design [竞品工作室] 2026"
WebSearch: "老虎机 SCATTER 图标 设计风格 竞品分析 [目标市场]"
WebSearch: "casino slot SCATTER symbol [主题关键词] art style visual design"
```

> 搜索词中的变量从已读取的 brief 数据填入：
> - [游戏主题] → brief.游戏基础.主题
> - [竞品工作室] → brief.参照与限制.竞品参照（如 Pragmatic Play / NetEnt 等）
> - [目标市场] → brief.游戏基础.目标市场

**必须获取的维度：**

| 维度 | 要求 |
|------|------|
| 竞品数量 | ≥ 3 个竞品的 SCATTER 图标特点 |
| 主体形态 | 宝石/宝箱/星形/人物等分布趋势 |
| 色彩偏好 | SCATTER 主色系（注意与WILD的差异化） |
| 发光特效 | 内发光/外发光/折射的竞品做法 |
| 文字处理 | 是否显示 "SCATTER" / "FREE SPINS" / 无文字 |
| 边框风格 | 圆形/方形等偏好 |

### 1.2 AI 模型能力搜索（与 wild-prompt 相同逻辑）

针对 Q_P1 选中的每个平台，各执行 3 条搜索：

```
WebSearch: "[平台名] image generation latest models [当前年份] features"
WebSearch: "[平台名] 最新图像生成模型 Prompt技巧 [当前年份]"
WebSearch: "[平台名] AI生图 模型能力 美术风格 参数 [当前年份]"
```

**每平台必须获取：**

| 维度 | 要求 |
|------|------|
| 模型版本 | ≥ 3 个可用版本号与发布日期 |
| 语言偏好 | 中文/英文/双语 |
| 结构方式 | 关键词/自然语言/结构化段落 |
| 关键参数 | 专属参数与语法 |
| 风格擅长 | 宝石/材质/发光类图像生成能力 |
| 局限性 | 已知弱点与规避方法 |

## Step 3：智能分析与选项生成

基于竞品分析 + AI 模型能力，为第三阶段问卷每题生成智能推荐选项。

**SCATTER 特有分析维度 → 选项生成映射：**

| 分析维度 | 输入数据 | 生成目标 |
|---------|---------|---------|
| 世界观氛围 | Q1(主题) + Q3(市场) + SCATTER竞品分析 | Q14 智能选项 |
| 美术风格 | Q_M1(主体类型) + Q12(复杂度) + AI模型擅长 | Q15 智能选项 |
| 材质与光照 | Q_M1主体类型 + 竞品分析 | Q16、Q17 智能选项 |
| 色彩（含色相差计算） | WILD主色（来自brief） + 竞品SCATTER色彩趋势 | Q19 智能选项（每项标注色相差） |
| SCATTER标签字体 | 游戏主题 + 竞品文字处理方式 | Q22_S 智能选项 |
| 边框形状 | Q_M1主体类型 + 竞品边框风格 | Q26_S 推荐偏向圆形/椭圆 |
| 框内背景 | Q_M1主体类型（宝石类→简洁背景）+ 竞品 | Q27_S 智能选项 |

**选项格式要求：**
- 每题 3-4 个选项
- 推荐项标注理由（"推荐 — SCATTER竞品趋势+AI模型擅长"）
- 色调选项额外标注与 WILD 主色的色相差数值
