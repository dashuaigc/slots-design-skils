# 核心美术约束（12条）

自检时逐条验证。文本可验证的 ❌ 项自动修正，仅生图后可验证的 ⚠️ 项记录到报告。详细自检表格见 `quality-control.md`（按需加载）。

1. **主体视角**：正面视角优先（front-facing view / 直视观众）；宝石/水晶/多面体类主体允许 3/4 视角或微俯视（以展示切割面与折射效果）
2. **主体占比**：占符号格高度 **65-85%**；视觉重心略高于画布中心（Y 轴 42-50%）
3. **主体超框**：**0-8% 可选**；宝石类超框部位为顶点/棱角；人物/神明类参考 WILD 规范（头部超框 3-8%）；超出部分仍在画布安全边距内
4. **安全边距**：四边 ≥ 5% 画布尺寸（外发光可超出，主体轮廓不可超出）
5. **视觉层级**：**SCATTER 视觉重量必须 ≥ 同套 WILD 图标**；80px 缩略图下 SCATTER 应是最先被注意到的图标；通过色相对比、外发光强度、饱和度实现超越，而非单纯靠主体尺寸
6. **色彩**：核心色 ≤ 3 种；主色饱和度 ≥ 65%（HSB S 值）；**与 WILD 主色色相差 ≥ 30°**（强制）；禁纯灰主体
7. **材质反光**：人物/神明皮肤与布料：哑光低光泽（specularity < 20%），皮肤次表面散射；**宝石/水晶/金属类允许高光折射和多点光源**（此规则例外于 WILD 皮肤规范）
8. **外发光**：必须存在；**强度不低于同套 WILD 图标的外发光**；禁纯白色；颜色与主色有色相关联；宝石类支持多点光源模拟切割面反光
9. **边框**：单线（Single Line），宽度 2-4% 画布；禁双线/厚框/多线并排；形状按 Q26_S 答案明确写入
10. **框内背景**：主题暗色氛围，强明暗对比；**宝石/水晶类允许使用极简深色单色底**（无需主题纹样，宝石本身足够）；禁花哨纹样、禁白色/亮色/高饱和背景
11. **SCATTER 标签文字位置**：**若 Q_S1 = 需要文字**：位于底部 1/4 区域（距顶 70-85%），向下超框 10-25%，横向两端各超框 2-5%；**若 Q_S1 = 不需要文字**：跳过此约束
12. **SCATTER 标签文字外观**：**若有文字**：3D 浮雕立体，颜色按 Q21_S 选择（金/银/紫/主题色，不默认金色）；有厚度侧面+高光面+阴影面至少 2 项；禁平面 2D；字体融入主题元素；字距非负值；64px 下可辨认文字内容
