### Batch 1.2｜SCATTER 主体设定

**Q_M1 SCATTER 主体物类别**
header: 主体物类别
multiSelect: false

  - 神明 / 神兽（神话神明、守护神兽、图腾造物等）
  - 器物 / 宝物（宝石、宝箱、神器、符文石、水晶球、徽章等）

**Q_M1b 主体具体身份设定**
header: 主体身份
multiSelect: false

### Q_M1c–Q_M1f 动态追问规则

**核心原则：** 根据 Q_M1 主体类别 + Q_M1b 具体身份，为每题**智能生成** 2-3 个高度贴合该身份的推荐选项。每道题必须提供至少 1 个具体的身份匹配推荐。

header: 体态动作

header: 面部气质
