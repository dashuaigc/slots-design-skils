# Nano Banana 生成参数问卷

## 弹窗通用规则

- 使用 `AskUserQuestion` 工具执行所有弹窗。
- 每次调用提交 1-4 个问题（按 Batch 分组）。
- 推荐选项放第一位并标注 `(Recommended)`。
- 用户可通过 Other 输入自定义值。

---

## Batch 1 — Prompt 来源

先扫描 `../../outputs/prompts/` 目录下所有 `nb2_*.md` 文件。

### 若找到文件

```
Q1: "请选择 Prompt 来源"
options:
  - 从已有文件选择（列出文件名）
  - 手动输入 Prompt（在 Other 中填写）
```

若选择已有文件且有多个文件，追加一题列出文件让用户选择具体文件。

### 若未找到文件

```
Q1: "未找到已有 nb2 Prompt 文件，请在 Other 中输入 Prompt"
options:
  - 输入 Prompt（在 Other 中填写）
```

### Prompt 语言自动选择规则

从 JSON 文件读取时，**自动**选择语言，无需弹窗：
1. 优先使用 `prompt_en`（英文）
2. 若无 `prompt_en`，使用 `prompt_cn`（中文）

## Batch 2 — 生成参数（3 题一组）

```
Q1: "画面比例？"
options:
  - 16:9（横版宽屏）(Recommended)
  - 1:1（正方形）
  - 4:3（横版经典）
  - 9:16（竖版宽屏）
  // Other 可输入：3:2, 2:3, 3:4, 5:4, 4:5, 21:9, 1:4, 4:1, 1:8, 8:1

Q2: "输出分辨率？"
options:
  - 1K（标准）(Recommended)
  - 2K（高清）
  - 4K（超高清）
  - 0.5K（快速预览）

Q3: "生成数量？"
options:
  - 1 张 (Recommended)
  - 2 张
  - 4 张
```

---

## 自动默认值（无需弹窗）

以下参数使用固定默认值，不再弹窗询问：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `google_search` | `true` | 始终开启文字搜索增强 |
| `google_image_search` | `true` | 始终开启图片搜索增强 |
| 参考图 | 无 / 用户主动提供 | 若用户在调用时附带了参考图则自动使用，否则默认无参考图 |

---

## 参数确认

选择完 Batch 2 后**直接开始生成**，不再弹窗确认。
生成过程中自动下载图片，无需额外确认。
