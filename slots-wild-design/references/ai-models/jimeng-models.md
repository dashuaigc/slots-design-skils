# 即梦（Jimeng）模型规范文档

## Seedream 5.0 Lite

### 基本信息
- 发布日期：2026年初
- 当前状态：最新版本
- 核心特性：联网图像生成（实时搜索参考）

### Prompt 能力
- 语言偏好：**中文优先**（支持自然语言描述）
- 结构方式：中文自然语言 + 标签组合，联网实时搜索参考图
- 最大长度：**800字**（含标点符号，严格限制）
- 支持参数：无专用参数，通过自然语言描述控制

### 美术风格
- 擅长风格：中式主题、风格化3D、卡通渲染、写实3D
- 不擅长：极度写实摄影风格、复杂光影渲染
- 质量特点：中文语义理解最佳，联网可抓取参考图提升准确度，国风题材积累丰富

### Prompt 构建技巧

**有效技巧：**
1. 中文自然语言优先：使用完整中文句子描述
2. 分段描述：主体 → 材质 → 光照 → 构图 → 细节
3. 负向提示有效：使用"禁止"、"不要"等词
4. 联网关键词有效：描述参考风格时可提及"类似xxx"、"参考xxx风格"

**常见误区：**
1. 过长Prompt：超过800字会被截断，需按优先级删减
2. 纯英文输入：中文模型英文理解弱，中文效果更好
3. 缺少负向限制：易生成不需要的复杂元素

### 字数控制策略

**当Prompt超过800字时，按以下优先级删减：**
1. 次要装饰细节描述 → 删减
2. 重复技术参数说明 → 删减
3. 负向排除词 → 保留最关键5条，其余删减
4. 次要光照描述 → 简化

## 即梦 4.5

### 基本信息
- 发布日期：2025年
- 当前状态：稳定版本，2K输出

### Prompt 能力
- 语言偏好：**中文优先**（支持中英双语）
- 结构方式：中文自然语言，可混用英文标签
- 最大长度：约1000字（比Seedream 5.0 Lite宽松）
- 支持参数：通过自然语言控制

### 美术风格
- 擅长风格：卡通3D、中式主题、喜庆题材
- 质量特点：2K分辨率输出，质感稳定，国风题材表现丰富

### Prompt 构建技巧

**有效技巧：**
1. 中文描述为主：主体、材质、光照用中文描述
2. 英文标签补充：风格关键词可用英文标签增强
3. 风格修饰词：`卡通3D`, `国风`, `高光`, `立体感`

**常见误区：**
1. 纯英文输入：中文模型对中文理解更好
2. 过度复杂句式：4.5版本对超长自然语言理解有限
