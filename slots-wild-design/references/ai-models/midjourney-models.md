# Midjourney 模型规范文档

## Midjourney V7

### 基本信息
- 发布日期：2025年6月
- 当前状态：默认版本（Default）

### Prompt 能力
- 语言偏好：英文（支持自然语言描述）
- 结构方式：自然语言句子 + 标签组合，可理解复杂场景描述
- 最大长度：建议 60-80 词，支持更长但需精炼
- 支持参数：`--v 7`, `--ar`, `--style raw`, `--no`, `--personalize`

### 美术风格
- 擅长风格：写实3D渲染、风格化3D、手绘插画、概念原画、动漫风格
- 不擅长：像素艺术（需特别提示）
- 质量特点：细节丰富、光影真实、材质质感优秀、手部细节改进

### 关键参数与语法

```
--v 7              使用V7版本（当前默认）
--ar [宽:高]        宽高比，如 --ar 16:9, --ar 2:3
--style raw        禁用MJ默认美学风格，使用原始Prompt描述
--no [负面词]     排除特定元素，多个词用逗号分隔
--personalize      开启个性化（需提前评分200张图片设置）
--q [2]           质量设置，默认--q 1，--q 2质量更高更慢
```

### Prompt 构建技巧

**有效技巧：**
1. 自然语言描述优先：使用完整句子描述主体、材质、光照、氛围
2. 细节分层描述：先主体，再材质，再光照，再构图
3. 负面提示有效：使用 `--no` 排除不需要的元素
4. 风格关键词有效：`stylized cartoon 3D`, `photorealistic`, `digital illustration`

**常见误区：**
1. 过度堆叠标签：V7 更擅长自然语言，标签过多反而效果差
2. 缺少负面提示：容易生成不需要的元素
3. 不指定宽高比：默认1:1，需明确指定
4. 忽略 `--style raw`：MJ会加入自身美学，可能与需求冲突

### 中英对照输出要求

- 主输入：英文 Prompt
- 对照参考：中文翻译（供用户理解，非用于生成）

## Midjourney V6.1

### 基本信息
- 发布日期：2024年7月30日
- 当前状态：稳定版本，比V6快25%

### Prompt 能力
- 语言偏好：英文（标签+短句组合）
- 结构方式：关键词堆叠 + 短句，对自然语言理解弱于V7
- 最大长度：建议 50-70 词
- 支持参数：`--v 6.1`, `--ar`, `--style raw`, `--no`

### 美术风格
- 擅长风格：写实3D、卡通3D、插画
- 质量特点：比V6更精准细节，生成速度快25%

### 关键参数

```
--v 6.1            使用V6.1版本
--ar [宽:高]        宽高比
--style raw        禁用默认美学
--no [负面词]     排除元素
```

### Prompt 构建技巧

**有效技巧：**
1. 标签优先：使用 `,,` 分隔的关键词
2. 短句补充：标签后加短句补充细节
3. 风格关键词：`cartoon 3D`, `PBR materials`, `volumetric lighting`

**常见误区：**
1. 过度自然语言：V6.1对长句理解弱于V7
2. 未指定风格：容易生成不一致风格

## Midjourney Niji 7

### 基本信息
- 发布日期：2026年1月9日
- 当前状态：动漫/插画风格最新版

### Prompt 能力
- 语言偏好：英文（动漫风格描述语）
- 结构方式：动漫风格关键词 + 自然语言
- 最大长度：建议 50-70 词
- 支持参数：`--niji 7`, `--ar`, `--style raw`, `--no`

### 美术风格
- 擅长风格：日系动漫插画、东方神明角色、风格化角色
- 不擅长：超写实、粗犷材质
- 质量特点：角色细节优秀，眼睛渲染突出，日中文本渲染更佳

### 关键参数

```
--niji 7           使用Niji 7版本
--ar [宽:高]        宽高比
--style raw        禁用默认美学
--no [负面词]     排除元素
```

### Prompt 构建技巧

**有效技巧：**
1. 动漫风格关键词：`anime style`, `manga`, `cel-shaded`, `vibrant colors`
2. 东方角色描述：强调眼部表情、服装细节
3. 风格修饰：`stylized character`, `expressive face`

**常见误区：**
1. 用写实描述：Niji偏向动漫，写实描述效果不如V7
2. 忽略文化语境：东方神明需适当描述发型、服饰
