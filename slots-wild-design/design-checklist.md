## 阶段二：生图后（图像评审）

> 对照 [wild-art-spec.md](wild-art-spec.md) 中的评分表进行评审。以下为快速核查项。

### 视觉层级 ｜ 规范第一章
- [ ] WILD 是卷轴中视觉重量最高的图标（或符合其定位层级）
- [ ] 64×64px 缩略图下仍能一眼认出是 WILD
- [ ] 主色调与 SCATTER/BONUS 有明显区分（色相差 ≥ 30°）

### 构图 ｜ 规范第二章
- [ ] 主体占画布面积的 55–75%
- [ ] 四边安全边距 ≥ 画布尺寸的 5%
- [ ] 主体视觉重心略高于中心（Y 轴约 45%）
- [ ] 主体完整，无裁切

### 色彩 ｜ 规范第三章
- [ ] 核心识别色不超过 3 种
- [ ] 主色饱和度（HSB S 值） ≥ 65%（刻意做旧风格除外）
- [ ] 主体最亮区域与卷轴背景对比度 ≥ 4:1（如背景已知）

### 发光 ｜ 规范第四章
- [ ] 存在外发光（Outer Glow）
- [ ] 外发光颜色与主色有色相关联（非纯白）
- [ ] 发光强度与风格匹配（写实风不过曝，插画风可夸张）

### 边框 ｜ 规范第五章
- [ ] 边框宽度在画布的 4–8%（如有边框）
- [ ] 边框四角视觉闭合（如有边框）
- [ ] 无边框时：轮廓有描边或边缘光与背景分离

### WILD 文字 ｜ 规范第六章
- [ ] 文字高度在画布的 12–25%（如有文字）
- [ ] 文字描边宽度 ≥ 字体大小的 4%（如有文字）
- [ ] 64×64px 下至少可辨认 3 个字母（如有文字）

### 技术规格 ｜ 规范第八章
- [ ] 格式为 PNG-32（带 Alpha 通道透明背景）
- [ ] 色彩空间为 sRGB
- [ ] 导出尺寸与 B1 中确认的规格一致
- [ ] Alpha 通道边缘平滑（无硬边/锯齿）
- [ ] 文件大小合理（512×512 建议 < 800KB）
- [ ] 文件命名符合规范：`wild_[主题]_[尺寸]_[状态].png`
